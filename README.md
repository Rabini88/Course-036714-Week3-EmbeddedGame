# Course 036714: Week 3 - Embedded Game

Welcome to the Embedded Systems Game Project! This repository contains the source code, hardware configuration, and design documentation for an ESP32-based embedded game, created as part of the "AI-Assisted Development" course.

## Overview
This project uses an ESP32 Development Board, a 0.96" OLED Display (SSD1306), tactile buttons, LEDs, and an active buzzer to create classic arcade-style games. 

## Documentation
Detailed documentation on hardware setup, connections, and the project blueprint can be found in the `docs/` folder:
- [Hardware Setup Guide](docs/Blair_Hardware_Setup_Guide.md)
- [AI Project Blueprint](docs/ai_project_blueprint.md)

## Development Setup
The project is configured to use PlatformIO in Visual Studio Code.
- Libraries required: `Adafruit SSD1306` and `Adafruit GFX`
